### Solution :

- Time Complexity :


```java
class MyStack
{
    int top;
    int arr[] = new int[1000];

    MyStack()
    {
        top = -1;
    }

    //Function to push an integer into the stack.
    void push(int a)
    {
        arr[++top]=a;
    } 

    //Function to remove an item from top of the stack.
    int pop()
    {
        if (top < 0)
        {
            return -1;
        }
        return arr[top--];
    }
}

```
